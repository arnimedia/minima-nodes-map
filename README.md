Minima Nodes Map
---

Minima Nodes Map, based on mapbox.com, deployed via Netlify.

[Web preview](https://nodes-minima.netlify.app/)

![Minima Nodes Map](minima_nodes_map.png)

### Run it

    npm install
    npm start

    open http://localhost:3000/

Runs the app from http://localhost:3000
